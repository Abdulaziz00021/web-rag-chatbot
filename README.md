 Web RAG Chatbot — AI-Powered Web Search Assistant

This project is an intelligent Web RAG (Retrieval-Augmented Generation) chatbot built using LangChain, Groq’s Llama 3.1, and Tavily Search API.
It can browse the web and generate real-time, paragraph-style answers — like a smarter AI-powered Google search assistant. 

 Technologies Used

Python 3.10+

Streamlit – Web App Interface

LangChain – RAG Framework

Groq LLM (Llama 3.1) – AI Model Backend

Tavily Search API – Real-time Web Search

 How It Works

1️⃣ User asks a query (e.g., “Latest news about AI regulations”)
2️⃣ Tavily fetches real-time search results
3️⃣ LangChain processes and embeds the context
4️⃣ Llama 3.1 (Groq) generates a clear, paragraph-style summary
5️⃣ Streamlit displays the answer beautifully in the UI

 Features

✅ Real-time information search
✅ Natural paragraph-style AI responses
✅ Context-aware reasoning (RAG pipeline)
✅ Simple Streamlit frontend
